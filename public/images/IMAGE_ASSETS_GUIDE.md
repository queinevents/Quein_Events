# Image Assets Guide - Dark Theme Aesthetic

This guide documents all image assets for the Quein Event Website with dark theme optimization.

## Overview

All images are optimized for dark backgrounds (#0A0A0A, #050505, #1A1A1A) with:
- Dramatic lighting and vibrant colors (purple #8B5CF6, blue #3B82F6, gold #F59E0B)
- WebP format with JPEG fallback for browser compatibility
- Compressed to specified sizes while maintaining quality
- Descriptive alt text for accessibility
- Responsive versions (2x, 3x) for retina displays

## Image Optimization Guidelines

### Compression
- Event category images: <200KB each
- Expertise images: <200KB each
- Team photos: <100KB each
- Testimonial photos: <50KB each
- Gallery images: <200KB each

### Format
- Primary: WebP for modern browsers
- Fallback: JPEG for older browsers
- Icons: SVG (optimized with SVGO)

### Responsive Versions
- 1x: Base size (mobile)
- 2x: Retina displays (tablet/desktop)
- 3x: High-density displays (premium devices)

### Dark Theme Optimization
- Ensure good contrast on dark backgrounds
- Add dark gradient overlays for text readability
- Use dramatic lighting and vibrant colors
- Test on actual dark backgrounds (#0A0A0A, #1A1A1A)

## Accessibility Requirements

### Alt Text Best Practices
- Describe what's in the image, not "image of" or "picture of"
- Keep concise but informative (<125 characters)
- Include context-specific details (event type, setting, key elements)
- Use empty alt text (alt="") for decorative images
- Ensure alt text is meaningful for screen reader users

### WCAG Compliance
- All images must have descriptive alt text
- Decorative images must have empty alt text
- Text overlays must have 4.5:1 contrast ratio
- Focus indicators must have 3:1 contrast ratio

## Image Sources

When sourcing real images:
1. Use high-quality event photography
2. Ensure proper licensing and permissions
3. Prefer images with dramatic lighting
4. Look for vibrant colors (purple, blue, gold accents)
5. Ensure images work well on dark backgrounds
6. Maintain consistent style across all images
