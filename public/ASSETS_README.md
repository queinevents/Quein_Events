## Replacement Instructions

### For Logo (logo.svg)
1. Export brand logo as SVG with transparent background
2. Recommended size: 200x60px or similar aspect ratio
3. Ensure crown element is prominent (brand identity)
4. Replace `public/images/logo.svg`

### For Service Icons
1. Create or export icons as SVG (64x64px recommended)
2. Use brand colors: purple (#8B5CF6), blue (#3B82F6), gold (#F59E0B)
3. Replace files in `public/icons/`

### For Portfolio Images
1. Collect high-quality photos from past events
2. Recommended size: 1200x900px (4:3 aspect ratio)
3. Optimize images (WebP format recommended)
4. Name descriptively (e.g., `corporate-conference-2024.jpg`)
5. Replace SVG placeholders in `public/images/portfolio/`

### For Favicon
1. Use a favicon generator tool:
   - [favicon.io](https://favicon.io/)
   - [realfavicongenerator.net](https://realfavicongenerator.net/)
2. Generate from logo.svg
3. Include multiple sizes: 16x16, 32x32, 48x48
4. Replace `app/favicon.ico`
